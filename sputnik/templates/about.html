About

I'm a big podcast junkie and this aggregator is built to fill my own needs, which are:

  • Work cross-platform (Linux, Firefox OS, Android, etc).
  • Personal feed that clearly displays the latest episodes and marks what has been listened to.
  • Requires low resources, i.e. small amount of JavaScript.
  • Little cruft and easy to use. No flashy JavaScript animations that would run poorly and load slowly on mobile.
  • Easy to leave, with RSS compatible export.

If you also love podcasts, please do yourself a service and sign up by clicking here. It's completely free.

Underlying technology

This website's back end is written in Python 3 with the web framework Django as a base. On top of that are a bunch of Python libraries and Django apps utilized.

As web server is Nginx being used and the application server uWSGI handles the Python part.

All of this is deployed on a VPS with the Linux distribution Debian.
